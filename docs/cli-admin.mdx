Admin-команды требуют прав редактора игры. Используйте их для создания и управления контентом квестов.

<Note>
  Для получения прав редактора на тестовом домене `tech.en.cx` напишите в [техподдержку сети](https://world.en.cx/UserDetails.aspx?uid=7).
</Note>

---

## Список авторских игр

### `admin-games`

Показывает игры, в которых вы являетесь автором или имеете права администратора.

```sh
encli admin-games
```

---

## Управление уровнями

### `admin-levels`

Показывает все уровни игры с их внутренними ID. ID нужны для команд переименования и создания бонусов.

```sh
encli admin-levels -game-id <id>
```

**Пример вывода:**

```
#   Level ID   Name
-   --------   ----
1   67890      Уровень 1
2   67891      Уровень 2
3   67892      Уровень 3
```

---

### `admin-level-content`

Читает полное содержимое уровня из панели администратора: текст задания, ответы секторов, бонусы, подсказки, настройки автоперехода и блокировки.

```sh
encli admin-level-content -game-id <id> <level-number>
```

**Пример:**

```sh
encli admin-level-content -game-id 12345 1
```

В выводе ID объектов указаны в квадратных скобках: `[bonus 111]`, `[sector 222]`, `[hint 333]`, `[task 444]`. Используйте эти ID в командах удаления и обновления.

---

### `admin-create-levels`

Создаёт указанное количество новых уровней в конце игры.

```sh
encli admin-create-levels -game-id <id> <count>
```

**Пример:**

```sh
encli admin-create-levels -game-id 12345 3
```

---

### `admin-delete-level`

Удаляет уровень по его номеру.

```sh
encli admin-delete-level -game-id <id> <level-number>
```

<Warning>
  Удаление уровня необратимо. Все бонусы, секторы, подсказки и задания уровня будут удалены вместе с ним.
</Warning>

---

### `admin-rename-level`

Переименовывает уровень. Используйте ID уровня из вывода `admin-levels`.

```sh
encli admin-rename-level -game-id <id> <level-id> <new-name>
```

**Пример:**

```sh
encli admin-rename-level -game-id 12345 67890 "Новое название"
```

---

### `admin-set-autopass`

Устанавливает таймер автоматического перехода на следующий уровень.

```sh
encli admin-set-autopass -game-id <id> <level-number> <HH:MM:SS> [penalty HH:MM:SS]
```

**Примеры:**

```sh
# Автопереход через 1 час 30 минут
encli admin-set-autopass -game-id 12345 1 1:30:00

# С штрафным временем 15 минут
encli admin-set-autopass -game-id 12345 1 1:30:00 0:15:00
```

---

### `admin-set-block`

Настраивает блокировку ответов: ограничивает количество попыток за период.

```sh
encli admin-set-block -game-id <id> <level-number> <attempts> <HH:MM:SS> [player]
```

**Примеры:**

```sh
# 3 попытки за 1 минуту (на команду)
encli admin-set-block -game-id 12345 1 3 0:01:00

# 3 попытки за 1 минуту на игрока
encli admin-set-block -game-id 12345 1 3 0:01:00 player
```

---

### `admin-swap-levels`

Меняет местами два уровня по их номерам.

```sh
encli admin-swap-levels -game-id <id> <level1> <level2>
```

**Пример:**

```sh
encli admin-swap-levels -game-id 12345 2 5
```

---

### `admin-insert-level`

Перемещает уровень на новую позицию (после указанного уровня; `0` — в начало).

```sh
encli admin-insert-level -game-id <id> <source-level> <after-level>
```

**Пример:**

```sh
# Переместить уровень 5 после уровня 2
encli admin-insert-level -game-id 12345 5 2
```

---

### `admin-clone-levels`

Создаёт N уровней, клонируя настройки существующего уровня.

```sh
encli admin-clone-levels -game-id <id> <count> <like-level>
```

**Пример:**

```sh
# Создать 3 уровня с настройками уровня 1
encli admin-clone-levels -game-id 12345 3 1
```

## Задания

### `admin-create-task`

Создаёт задание (текст задания) на уровне.

```sh
encli admin-create-task -game-id <id> <level-number> <text>
```

**Пример:**

```sh
encli admin-create-task -game-id 12345 1 "Найдите место, где встречаются три дороги..."
```

---

### `admin-update-task`

Обновляет текст задания по его ID. ID берите из вывода `admin-level-content` (`[task <id>]`).

```sh
encli admin-update-task -game-id <id> <level-number> <task-id> <text>
```

---

### `admin-delete-task`

Удаляет задание по его ID.

```sh
encli admin-delete-task -game-id <id> <level-number> <task-id>
```

---

### `admin-set-comment`

Устанавливает название и комментарий уровня (комментарий виден только организаторам).

```sh
encli admin-set-comment -game-id <id> <level-number> <name> [comment]
```

**Пример:**

```sh
encli admin-set-comment -game-id 12345 1 "Старый город" "Уровень у фонтана, проверить GPS"
```

---

## Бонусы

### `admin-create-bonus`

Создаёт бонус на уровне с одним или несколькими принимаемыми ответами.

```sh
encli admin-create-bonus -game-id <id> <level-num> <level-id> <name> <answer1> [answer2 ...]
```

<Note>
  `level-id` — внутренний ID уровня из вывода `admin-levels`, не номер уровня.
</Note>

**Пример:**

```sh
encli admin-create-bonus -game-id 12345 1 67890 "Бонус 1" "ответ1" "ответ2"
```

---

### `admin-update-bonus`

Обновляет поля бонуса. Поддерживаемые ключи: `name`, `task`, `hint`, `answers` (через запятую).

```sh
encli admin-update-bonus -game-id <id> <level-num> <bonus-id> <key=value ...>
```

**Пример:**

```sh
encli admin-update-bonus -game-id 12345 1 111 name="Новое имя" answers="код1,код2"
```

---

### `admin-delete-bonus`

Удаляет бонус по его ID. ID берите из вывода `admin-level-content` (`[bonus <id>]`).

```sh
encli admin-delete-bonus -game-id <id> <level-number> <bonus-id>
```

## Подсказки

### `admin-create-hint`

Создаёт подсказку на уровне с задержкой перед открытием.

```sh
encli admin-create-hint -game-id <id> <level-number> <delay HH:MM:SS> <text>
```

**Пример:**

```sh
# Подсказка откроется через 30 минут
encli admin-create-hint -game-id 12345 1 0:30:00 "Ищите у старого фонаря"
```

---

### `admin-update-hint`

Обновляет поля подсказки. Поддерживаемые ключи: `text`, `delay` (в формате `HH:MM:SS`).

```sh
encli admin-update-hint -game-id <id> <level-num> <hint-id> <key=value ...>
```

**Пример:**

```sh
encli admin-update-hint -game-id 12345 1 333 text="Новый текст" delay=0:45:00
```

---

### `admin-delete-hint`

Удаляет подсказку по её ID. ID берите из вывода `admin-level-content` (`[hint <id>]`).

```sh
encli admin-delete-hint -game-id <id> <level-number> <hint-id>
```

## Сообщения

### `admin-messages`

Показывает сообщения уровня с их ID и привязками.

```sh
encli admin-messages -game-id <id> <level-number>
```

---

### `admin-create-message`

Создаёт игровое сообщение. Используйте `level-id` (внутренний ID из `admin-levels`).

```sh
encli admin-create-message -game-id <id> <level-id> <text>
```

**Пример:**

```sh
encli admin-create-message -game-id 12345 67890 "Внимание! Изменение маршрута."
```

---

### `admin-update-message`

Обновляет сообщение. Поддерживаемые ключи: `text`, `required_points`, `mode`, `levels`.

```sh
encli admin-update-message -game-id <id> <level-number> <message-id> <key=value ...>
```

**Пример:**

```sh
encli admin-update-message -game-id 12345 1 555 text="Новый текст" mode=chosen levels=67890
```

---

### `admin-delete-message`

Удаляет сообщение по его ID.

```sh
encli admin-delete-message -game-id <id> <level-number> <message-id>
```

## Команды и коррекции времени

### `admin-teams`

Показывает список команд, зарегистрированных в игре.

```sh
encli admin-teams -game-id <id>
```

---

### `admin-corrections`

Показывает список начислений бонусного/штрафного времени.

```sh
encli admin-corrections -game-id <id>
```

---

### `admin-add-correction`

Добавляет начисление времени для команды. Уровень `0` применяется ко всем уровням.

```sh
encli admin-add-correction -game-id <id> <team> <bonus|penalty> <HH:MM:SS> <level> <comment>
```

**Пример:**

```sh
# Бонус 10 минут команде "Команда А" за красоту
encli admin-add-correction -game-id 12345 "Команда А" bonus 0:10:00 0 "за красоту"
```

---

### `admin-delete-correction`

Удаляет начисление по его ID.

```sh
encli admin-delete-correction -game-id <id> <correction-id>
```

## Настройки игры

### `admin-game-info`

Показывает информацию об игре: название, авторы, описание, приз, дата финиша.

```sh
encli admin-game-info -game-id <id>
```

---

### `admin-update-game`

Обновляет настройки игры. Передавайте только те поля, которые нужно изменить.

```sh
encli admin-update-game -game-id <id> <key=value ...>
```

Поддерживаемые ключи: `title`, `authors`, `description`, `prize`, `finish` (формат `DD.MM.YYYY HH:MM:SS`).

**Пример:**

```sh
encli admin-update-game -game-id 12345 title="Новое название" description="Описание игры"
```

## Копирование и очистка игры

### `admin-wipe-game`

Полностью обнуляет игру: удаляет все уровни, бонусы, секторы, подсказки и коррекции. Игра остаётся пустой оболочкой.

```sh
encli admin-wipe-game -game-id <id>
```

<Warning>
  Операция необратима. Рекомендуется использовать перед `admin-copy-game` для чистого копирования.
</Warning>

---

### `admin-copy-game`

Копирует всю игру (уровни, настройки, бонусы, секторы, подсказки) из исходной игры в целевую.

```sh
encli admin-copy-game -game-id <source-id> <target-id>
```

**Пример:**

```sh
# Сначала очистить целевую игру
encli admin-wipe-game -game-id 67890

# Затем скопировать
encli admin-copy-game -game-id 12345 67890
```

## Жизненный цикл игры

| Команда | Описание |
|---|---|
| `admin-deliver` | Пометить игру как состоявшуюся |
| `admin-not-deliver` | Пометить игру как несостоявшуюся |
| `admin-award-points` | Начислить очки участникам |
| `admin-end-ratings` | Завершить приём оценок |
| `admin-calc-ik` | Рассчитать игровой коэффициент (ИК) |
| `admin-action-monitor` | Показать монитор действий в игре |

```sh
encli admin-deliver -game-id 12345
encli admin-award-points -game-id 12345
encli admin-end-ratings -game-id 12345
encli admin-calc-ik -game-id 12345
```

## Типичный рабочий поток организатора

```sh
# 1. Посмотреть список своих игр
encli admin-games

# 2. Создать уровни
encli admin-create-levels -game-id 12345 5

# 3. Посмотреть уровни с ID
encli admin-levels -game-id 12345

# 4. Добавить задание на уровень 1
encli admin-create-task -game-id 12345 1 "Текст задания..."

# 5. Добавить сектор
encli admin-create-sector -game-id 12345 1 "Сектор А" "КОД1" "код1"

# 6. Добавить бонус (нужен level-id из admin-levels)
encli admin-create-bonus -game-id 12345 1 67890 "Бонус 1" "БОНУС1"

# 7. Добавить подсказку (откроется через 30 мин)
encli admin-create-hint -game-id 12345 1 0:30:00 "Подсказка..."

# 8. Проверить содержимое уровня
encli admin-level-content -game-id 12345 1
```

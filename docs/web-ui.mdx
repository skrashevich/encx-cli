`encli` включает встроенный локальный Web UI — чат-интерфейс для работы с LLM-агентом прямо в браузере. Запускается флагом `-web`.

## Запуск

```sh
export LLM_API_KEY=sk-or-v1-...
export LLM_MODEL=anthropic/claude-sonnet-4

encli -web
# Открывается http://127.0.0.1:8787
```

Браузер открывается автоматически. Для остановки нажмите `Ctrl+C`.

### Адрес и порт

По умолчанию Web UI слушает на `127.0.0.1:8787`. Изменить адрес можно флагом или переменной окружения:

```sh
# Флаг
encli -web -web-addr 0.0.0.0:8787

# Переменная окружения
ENCLI_WEB_ADDR=0.0.0.0:8787 encli -web
```

<Warning>
  При использовании `0.0.0.0` Web UI будет доступен из локальной сети. Убедитесь, что это соответствует вашим требованиям безопасности.
</Warning>

## Интерфейс

### Общий вид

Web UI состоит из трёх зон:

- **Боковая панель** — история чатов с поиском, выбор домена и игры, кнопка нового чата.
- **Шапка** — текущая LLM-модель, переключатель режима агента, переключатель темы (◐).
- **Область чата** — история сообщений, поле ввода, лог вызовов инструментов.

### Авторизация

Авторизация выполняется прямо в интерфейсе: введите домен, логин и пароль. Сессия сохраняется в `~/.config/encli/<domain>.json` и используется повторно при следующем запуске.

### Выбор игры

В боковой панели выберите домен и игру из выпадающего списка. Каждый чат привязан к конкретной игре.

---

## Режимы безопасности агента

В шапке UI есть переключатель режима безопасности. Он применяется к каждому чату независимо.

<Tabs>
  <Tab title="Только чтение">
    Агент может только читать данные. Все инструменты, изменяющие игру, заблокированы.

    Используйте для аудита и просмотра содержимого без риска случайных изменений.
  </Tab>
  <Tab title="С согласованием">
    Перед каждым изменяющим действием UI показывает запрос подтверждения. Вы можете принять или отклонить каждое действие.

    Режим по умолчанию для новых чатов.
  </Tab>
  <Tab title="Полный доступ">
    Агент выполняет все действия без подтверждения.

    Используйте, когда доверяете агенту и хотите ускорить работу.
  </Tab>
</Tabs>

Режим можно изменить в любой момент — он применяется к следующему сообщению.

## История чатов

Все чаты сохраняются на диске в `~/.config/encli/web/chats/`. При следующем запуске история восстанавливается автоматически.

### Поиск по истории

В боковой панели есть поле поиска — ищет по заголовку чата, домену и содержимому сообщений.

### Экспорт чата

Каждый чат можно экспортировать в Markdown или JSON через кнопку в шапке чата.

---

## Выбор модели

Текущая модель отображается в левом верхнем углу шапки. Модель берётся из переменной окружения `LLM_MODEL` (или `OPENROUTER_MODEL`).

Для смены модели перезапустите `encli -web` с другой переменной:

```sh
LLM_MODEL=google/gemini-2.5-pro-preview encli -web
```

---

## Тёмная тема

Переключатель темы (◐) находится в правом верхнем углу шапки. Выбор сохраняется в браузере.

---

## Запуск с начальными параметрами

Можно передать домен, игру и режим безопасности при запуске:

```sh
encli -web \
  -domain tech.en.cx \
  -game-id 12345 \
  -readonly
```

Эти значения используются как значения по умолчанию для новых чатов.

## Docker

При запуске в Docker пробросьте порт и передайте переменные окружения:

```sh
docker run --rm -p 8787:8787 \
  -e LLM_API_KEY=sk-or-v1-... \
  -e LLM_MODEL=anthropic/claude-sonnet-4 \
  -e ENCLI_WEB_ADDR=0.0.0.0:8787 \
  ghcr.io/skrashevich/encx-cli -web
```

<Note>
  В Docker история чатов не сохраняется между запусками контейнера, если не примонтировать том для `~/.config/encli/`.
</Note>

## API Web UI

Web UI использует внутренний REST API. Основные эндпоинты:

| Метод | Путь | Описание |
|---|---|---|
| `GET` | `/api/v1/chats` | Список чатов |
| `POST` | `/api/v1/chats` | Создать чат |
| `GET` | `/api/v1/chats/{id}` | Получить чат |
| `DELETE` | `/api/v1/chats/{id}` | Удалить чат |
| `POST` | `/api/v1/chats/{id}/messages` | Отправить сообщение |
| `GET` | `/api/v1/chats/{id}/events` | SSE-поток событий |
| `POST` | `/api/v1/chats/{id}/cancel` | Отменить выполнение |
| `GET` | `/api/v1/chats/{id}/approval` | Получить запрос на подтверждение |
| `POST` | `/api/v1/chats/{id}/approval` | Ответить на запрос подтверждения |
| `GET` | `/api/v1/chats/{id}/export` | Экспортировать чат (json/markdown) |
| `POST` | `/api/v1/auth/login` | Авторизоваться |
| `POST` | `/api/v1/auth/logout` | Выйти |
| `GET` | `/api/v1/auth/status` | Статус авторизации |
| `GET` | `/api/v1/catalog/domains` | Список авторизованных доменов |
| `GET` | `/api/v1/catalog/games` | Список игр на домене |
| `GET` | `/api/v1/agent/config` | Конфигурация агента (модель, base URL) |
